# Sales Performance Analysis

## Tools Used
- Excel

## Objective
To analyze sales data and identify top-performing products and categories.

## Dataset
The dataset contains order details including product, category, quantity, price, and total sales.

## Key Insights
- Electronics category generated the highest sales
- Laptop and Mobile are the top-selling products
- Clothing category has the lowest sales
- Higher quantity directly increases revenue

## Files in This Project
- sales_data.xlsx – Raw sales data
- sales_by_category.png – Sales by category chart

## Conclusion
This project demonstrates basic data analysis skills using Excel such as formulas, pivot tables, and charts.
